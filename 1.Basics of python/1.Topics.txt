What is a Programming Language?

A programming language is a formal language that
provides instructions for a computer to execute. It's
how humans communicate with computers to create
software, applications, and other technology.

High Level Programming Language vs Low Level Programming Language

High level programming languages are those that are
written in a human-readable format and are
translated into machine code by a compiler or interpreter.

Low level programming languages are those that are
written in a machine-readable format and are
translated into machine code by a compiler or interpreter.

Compiler vs Interpreter

A compiler is a program that translates source code
into machine code at once, which can be executed by a computer.

An interpreter is a program that translates source code
into machine code Line By Line, which can be executed by a computer.
